## 1.0 — Initial public release

The first public release of AI Boost for Joomla: a complete SEO & AEO package for Joomla 5 and 6, available as a Free edition plus an optional Pro Upgrade.

### SEO

- **Schema.org structured data** — Organisation/business JSON-LD for dozens of site types (LocalBusiness, Restaurant, Hotel, MedicalClinic, LegalService, Store, Person, NewsMediaOrganization and more), WebSite schema with SearchAction, Article schema, opening hours and breadcrumbs. *Pro adds:* FAQ/QAPage (auto-detect + manual builder), HowTo, Event and Author Entity schema, extended business details, and Services & Prices.
- **XML sitemap** — dynamic sitemap for articles, categories and menu items with change-frequency and priority controls, plus a live preview with validation in the admin panel. *Pro adds:* sitemap index, image sitemap, hreflang alternates, tag URLs, per-type priorities and a Google News sitemap.
- **Technical SEO** — page title and meta description templates per page type, canonical URL management, 404 logging and a conflict-resolution mode that cooperates with other SEO extensions.
- **Social Meta / OpenGraph** — OpenGraph and Twitter Card tags with default share image. *Pro adds:* per-article OG overrides via Joomla custom fields, `og:locale` and Twitter site handle.
- **Analytics & Tracking** *(Pro)* — Google Analytics 4 (with GDPR consent modes), Google Tag Manager, site verification (Google Search Console, Bing, Facebook) and Meta Pixel with standard and custom conversion events.

### AI Visibility (AEO)

- **llms.txt** — an AI-readable site summary served at `/llms.txt`, with configurable description, recent articles and custom pages. *Pro adds:* `llms-full.txt` full-site index.
- **Markdown pages** — any page can be served as clean Markdown for AI agents (`.md` suffix, `?markdown=1`, or `Accept: text/markdown`).
- **AI signals** — AI-oriented meta tags.
- **Crawlers & Robots** — managed `robots.txt` block with per-bot allow/block rules for AI crawlers (GPTBot, ClaudeBot, PerplexityBot, Google-Extended and more), SEO scraper blocking and free-form custom rules.
- **IndexNow** *(Pro)* — instant URL submission to IndexNow-enabled search engines on publish/update, with one-click API key generation.

### Tools & admin experience

- **Autopilot** — a guided setup checklist (Site Identity → Schema.org Core → Sitemap → Social Meta) with live completion tracking.
- **Health** — a 0–100 health score with per-check fix actions that jump straight to the responsible setting; includes duplicate-tag and plugin-conflict scanning and an error log.
- **Redirects** — 301/302/303/307/308 redirect manager with hit counters, 404 log and CSV import.
- **Analyzers & URL Checker** — on-page SEO analyzer, JSON-LD validator and a batch URL checker fed from your sitemap.
- **Import / Export** — settings backup and restore as JSON.
- **Multilingual output** *(Pro)* — per-language values for schema and OpenGraph fields, hreflang alternates in the XML sitemap and Falang integration. The admin interface itself is English.
- **Custom Code** *(Pro)* — head/body/footer code injection, scopable to specific menu items.

### Licensing

- Pro is a yearly subscription in three plans that differ only in site count (3 / 10 / unlimited), sold through Lemon Squeezy.
- **Perpetual activation:** once verified, Pro features keep working on that site forever — licence expiry only pauses updates and support, and activation survives uninstall/reinstall. See [Licence & Plans](license-plans.md).
- The Free edition shows Pro features as visible locked cards with an **Upgrade to Pro** button — nothing is hidden.
